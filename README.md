### LabVIEW is a development environment and platform for running programs written in the National Instruments "G" graphical programming language.

## Results of working with the program
Programs use string functions: String Length, Concatenate Strings, String Subset, Match Pattern, Format Into String, Scan From String. As well as features of tables and various sorting methods.
